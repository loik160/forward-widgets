# Forward Widgets

我的 [Forward](https://apps.apple.com/app/id6443475928) Widget 合集。

## 订阅地址

在 Forward App 中添加以下订阅地址：

```
https://raw.githubusercontent.com/loik160/forward-widgets/main/widgets.fwd
```
